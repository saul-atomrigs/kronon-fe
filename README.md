# 실행 방법
1. 레포지토리를 클론합니다

```
git clone https://github.com/saul-atomrigs/kronon-fe-binance-clone.git
```

2. 로컬에서 실행합니다

```
yarn dev
```

3. 브라우저에서 로컬호스트 3000포트로 진입한 후, http://localhost:3000/en/trade/BTCUSDT 로 이동합니다.
